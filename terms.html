HTML TAGS

What is HTML?

HTML, or Hypertext Markup Language, is a "language" for creating web pages. It is consisted of elements and tags that tell your internet browser how to display the content on the web page.

What are tags?

Tags are ways of formatting and displaying content, like headings, pictures, and much more. There are two kinds of tags: starting tags and ending tags. They are very similar, except ending tags have a / at the beginning.

An element is a starting and ending tag, with content inbetween.

<a>
Defines a hyperlink (or link), can link from one page to another.
<article>
Specifies independent or self-contained content; News stories, blog or forum posts.
<aside>
Defines some content aside form the content it is placed in.
<audio>
Embeds sound content in a document, like music or other audio streams.
<b>
Specifies bold text.
<blockquote>
Specifies a section quoted from another source.
<body>
Defines a document's body.
<br>
Inserts a single line break.
<caption>
Defines a caption, usually for a table.
<dd>
DESCRIBES a term/name in a description list. This text is in a "<dd>"!
<div>
Defines a division or section in an HTML document, often used a sort of "container".
<dl>
Defines a description list.
<dt>
DEFINES a term/name in a description list.
<em>
Defines emphasized text, displayed in italics.
<embed>
Defines a container for an external resource, like a media player or plug-in application.
<figcaption>
Defines a caption for a figure.
<figure>
Specifies self-contained content, like photos, illustrations, etc.
<footer>
Defines a footer for a document/section, which typically contains copyright info, related documents, etc.
<form>
Creates an HTML form for user input.
<h1>, <h2>, <h3>, <h4>, <h5>, <h6>
Define HTML headings, with h1 defining most important and h6 defining least important.
<head>
A container for metadata (data about data), placed between html tag and body tag.
<header>
Represents a container for introductory content or a set of navigational links.
<hr>
Horizontal line often used to separate content in an HTML page.

Keep an eye out for more terms!

CSS

What is CSS?

CSS, or Cascading Style Sheets, describes how HTML elements should be displayed. It is used to stylize web pages. It does this by assigning styles or behaviors to elements, like color, border, margin, or font style.

background-color
Property that sets background color of an element.
border-style
Sets the style of an elements borders, like a solid or dashed border.
font-family
Specifies the font for an element.
@import
A rule that allows you to import a style sheet into another style sheet.
margin
Sets the margins for an element.
